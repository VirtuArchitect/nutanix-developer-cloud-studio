# v0.8.0-prism-readonly-inventory

`v0.8.0-prism-readonly-inventory` adds the first production-shaped Prism Central inventory boundary while keeping all live calls and mutation operations disabled.

## Highlights

- Added a Prism read-only inventory adapter contract.
- Added mock and disabled-real Prism adapter implementations.
- Added API-backed Prism inventory import guarded by reachable NCI integration configuration.
- Imported mock Prism records for clusters, projects, images, networks, categories, and VMs.
- Mapped imported image records into draft AHV image profile candidates.
- Added Admin Providers UI for import evidence, scope, blocked operations, and inventory records.
- Expanded unit, API, client, and Playwright smoke coverage.

## Important Notes

No real Prism Central API call is made in this release. The real adapter path returns disabled evidence and does not contact external endpoints. Create, update, delete, clone, power, resize, and category mutation operations remain blocked.
