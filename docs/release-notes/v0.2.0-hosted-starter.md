# v0.2.0-hosted-starter

`v0.2.0-hosted-starter` moves Nutanix Developer Cloud Studio from a purely clickable MVP into a hosted/on-prem starter prototype with API-backed workflow state.

## Highlights

- API-backed approval queue for AI endpoint and regulated-style requests.
- Approve/reject API decisions that update mock environment state and audit history.
- API-backed environment detail view with metadata, jobs, approvals, and audit events.
- Dashboard refresh toward an operations/on-prem console layout.
- Admin integration readiness panel connected to API data.
- Playwright smoke coverage for approval queue and environment detail.

## Notes

This is still an independent prototype. All integrations, provisioning jobs, approvals, costs, and environments are simulated for demonstration purposes only. It does not currently provision real Nutanix infrastructure.
