# v2.80.0 - State And Backup Hardening

## Summary

Hardened JSON prototype state backup and restore with integrity manifests.

## Added

- Backup manifests with schema version, source path, timestamp, file size, SHA-256 checksum, and guardrail notes.
- Restore-time manifest validation.
- Backup/restore smoke coverage for manifest creation and checksum validation.
- `npm run validate:backup`.

## Guardrails

- This remains prototype JSON state tooling.
- Production deployments still require a durable database and enterprise backup process.
