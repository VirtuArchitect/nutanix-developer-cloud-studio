# v3.9.0 - Lab Connectivity Preflight

Added a dry-run connectivity preflight for controlled read-only lab enablement.

## Added

- `GET /api/prism/lab-connectivity-preflights`.
- `POST /api/prism/lab-connectivity-preflights`.
- Validation for hardened profile evidence, TLS mode, CA reference, timeout/retry policy, credential resolver readiness, and read-only endpoint allowlists.

## Safety

- The preflight does not send live requests or import inventory.
