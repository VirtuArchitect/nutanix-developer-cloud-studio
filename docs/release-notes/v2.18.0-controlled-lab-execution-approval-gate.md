# v2.18.0-controlled-lab-execution-approval-gate

Release date: 2026-07-07

This release adds evidence-only controlled lab execution approval gates. Approval gates link to proposal exports and record the final human decisions that must exist before any future controlled lab execution phase can be considered.

## Added

- Controlled lab execution approval records linked to proposal exports.
- API endpoints for listing and recording execution approval gates.
- Admin Operations approval gate panel.
- Platform owner, security reviewer, lab owner, rollback owner, and executive sponsor decisions.
- Tests proving missing approvals block advancement and real adapter execution remains disabled.

## Verification

- `npm.cmd run test -- server/controlledLabExecutionApproval.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Full phase gate: `.\scripts\run-phase-gate.ps1 -TargetPhase v2.18.0-controlled-lab-execution-approval-gate`

## Security Notes

- Approval gates are evidence-only.
- `provisioningEnabled=false` remains enforced.
- Real Nutanix adapter execution remains disabled.
- Approval records store decisions and evidence references only, not provider data, credentials, tokens, or secrets.
