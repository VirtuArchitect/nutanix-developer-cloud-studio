# v2.19.0-controlled-lab-execution-rehearsal-packet

Release date: 2026-07-07

This release adds evidence-only controlled lab execution rehearsal packets. Packets freeze the approved execution evidence before any future live-lab adapter operation can be considered.

## Added

- Controlled lab execution rehearsal packet records linked to approval gates.
- API endpoints for listing and preparing rehearsal packets.
- Admin Operations rehearsal packet panel.
- Frozen runbook, rollback owner, emergency contact, stop condition, proposal export, audit export, and approval evidence references.
- Tests proving missing approval gates or incomplete frozen evidence block packet readiness.

## Verification

- `npm.cmd run test -- server/controlledLabExecutionRehearsalPacket.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Full phase gate: `.\scripts\run-phase-gate.ps1 -TargetPhase v2.19.0-controlled-lab-execution-rehearsal-packet`

## Security Notes

- Rehearsal packets are evidence-only.
- `provisioningEnabled=false` remains enforced.
- Real Nutanix adapter execution remains disabled.
- Packet records freeze evidence references only, not provider data, credentials, tokens, or secrets.
