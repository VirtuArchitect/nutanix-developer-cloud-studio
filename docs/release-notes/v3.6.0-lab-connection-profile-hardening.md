# v3.6.0 - Lab Connection Profile Hardening

Added hardening records for read-only lab connection profiles.

- `GET /api/prism/read-only-lab-profile-hardening`.
- `POST /api/prism/read-only-lab-profile-hardening`.

- No Prism Central network call is made.
