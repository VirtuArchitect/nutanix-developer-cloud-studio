# v0.3.0-integration-readiness

`v0.3.0-integration-readiness` prepares Nutanix Developer Cloud Studio for a more credible hosted/on-prem evaluation by adding mocked identity, role context, integration configuration, and readiness checks.

## Highlights

- Mock OIDC session and role model for developer, approver, and platform admin flows.
- API-backed integration configuration records for lab endpoint and credential profile placeholders.
- API-backed readiness checks for NCI, NKP, NDB, NUS, NCM, and NAI planning.
- Admin access model and integration configuration panels.
- Dashboard refresh that prioritizes environment operations over a large marketing-style hero.
- Database-ready `ApiRepository` persistence contract.
- Hosted starter validation script.
- Expanded `.env.example` and on-prem deployment notes.

## Notes

This remains an independent prototype. Auth, roles, readiness checks, integrations, approvals, costs, and provisioning are simulated and do not call real Nutanix services.
