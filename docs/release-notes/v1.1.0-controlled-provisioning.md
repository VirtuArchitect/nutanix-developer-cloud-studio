# v1.1.0-controlled-provisioning

`v1.1.0-controlled-provisioning` adds the first operator-controlled promotion gate for the AHV VM sandbox path.

## What Changed

- Added controlled provisioning gate records attached to VM sandbox dry-run plans.
- Added API endpoints for gate review request, approval, rejection, and listing.
- Added checks for dry-run validation, rollback readiness, destroy readiness, manual approval, authorized lab scope evidence, and mutation kill switch state.
- Added Admin Control Plane UI for gate review and decision handling.
- Added audit events and automated smoke coverage for the gate workflow.

## Safety Boundary

This release does not provision AHV resources. Approval records operator intent and readiness only. `provisioningEnabled` remains `false` until a future authorized adapter phase adds a real create/destroy implementation with lab scope and pentest evidence.
