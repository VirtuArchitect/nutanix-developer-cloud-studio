# v2.3.0-adapter-enable-contract-hardening

`v2.3.0-adapter-enable-contract-hardening` adds the explicit review contract required before any future real Nutanix adapter can move beyond disabled preflight evidence.

## Added

- Adapter enablement evidence records for NCI, NKP, NDB, NUS, NCM, and NAI.
- `/api/adapter-enablement/records` endpoints for listing and recording Platform Admin reviews.
- Checks for approved lab scope, credential reference diagnostics, provider readiness, adapter readiness, audit export readiness, rollback ownership, and real-adapter disabled state.
- Admin Providers panel showing the latest adapter enablement contract, blocked operations, and evidence summary.
- Unit/API/client and Playwright smoke coverage for the contract workflow.

## Security Notes

This release does not enable real provisioning. A real-adapter switch set through environment variables fails the v2.3 contract check because this phase is review-only. Credential values remain outside NDC Studio, and mutation operations stay blocked.
