# v4.4.0 - Pilot Evidence Review Queue

This release adds reviewer workflow records for pilot evidence.

## Added

- `GET /api/prism/pilot-evidence-reviews`
- `POST /api/prism/pilot-evidence-reviews`
- Admin Providers panel for evidence reviews.
- Reviewer, approval/rejection decision, findings, checks, and evidence fields.

## Safety

- Reviews cannot enable live HTTP.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
