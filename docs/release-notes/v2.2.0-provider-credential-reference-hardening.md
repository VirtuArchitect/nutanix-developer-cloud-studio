# v2.2.0-provider-credential-reference-hardening

Release date: 2026-07-06

## Summary

This release hardens provider credential references before any live Nutanix adapter can be configured. Provider configuration now validates credential profile references, exposes diagnostics, and rejects inline access material.

## Added

- Credential reference validator.
- `/api/provider-credentials/diagnostics` endpoint.
- Admin Providers credential reference diagnostics panel.
- Phase gate validation for provider credential references.
- Server, client, and E2E coverage for the reference boundary.

## Guardrails

- Credential values remain outside NDC Studio.
- Integration config stores reference names only.
- Real Nutanix mutation adapters remain disabled.
