# v3.3.0 - Real Read-Only Adapter Interface

Added the disabled real read-only adapter interface contract for Prism Central inventory reads.

## Added

- `GET /api/prism/real-read-only/adapter-interfaces`.
- `POST /api/prism/real-read-only/adapter-interfaces`.
- Prism Central list endpoint mapping for clusters, projects, images, subnets, categories, and VMs.

## Safety

- Adapter execution remains disabled.
- Credential resolution and network calls remain blocked.
