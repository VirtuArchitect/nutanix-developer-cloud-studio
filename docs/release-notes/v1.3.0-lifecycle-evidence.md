# v1.3.0-lifecycle-evidence

`v1.3.0-lifecycle-evidence` adds the evidence layer needed before controlled provisioning or platform-service flows can move beyond planning.

## What Changed

- Added lab authorization scope records with approved project, cluster, network, test window, allowed actions, excluded actions, and pentest scope evidence.
- Added VM lifecycle proof records for controlled gate, rollback, and destroy verification.
- Added API endpoints for creating and listing lab authorization scopes and lifecycle proofs.
- Added Admin Control Plane UI for recording and reviewing evidence.
- Connected controlled provisioning gates to active lab authorization scope evidence.
- Connected platform-service planning to recorded VM lifecycle proof.

## Safety Boundary

This release records evidence only. It does not create, roll back, or destroy AHV resources, and it does not call NKP, NDB, NUS, or NAI APIs. `provisioningEnabled` remains `false`.
