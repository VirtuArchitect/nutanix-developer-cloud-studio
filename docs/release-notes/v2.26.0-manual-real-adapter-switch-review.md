# v2.26.0-manual-real-adapter-switch-review

Release date: 2026-07-07

## Summary

This release adds manual real-adapter switch reviews. Reviews link lab scope activations to named switch operator, second reviewer, maintenance window, switch-state audit references, and rollback contact evidence before any manual switch change can be considered outside the prototype.

## Added

- Manual real-adapter switch review records linked to lab scope activations.
- API endpoints for listing and recording switch reviews.
- Admin Operations switch review panel.
- Switch operator, second reviewer, maintenance window, switch-state audit, rollback contact, and disabled switch checks.
- Tests proving missing activations or incomplete switch review evidence block readiness.

## Security Notes

- Switch reviews are evidence-only records.
- `provisioningEnabled` remains `false`.
- Provider kill-switch state remains disabled.
- Switch review APIs require the `Platform Admin` role.
- The prototype does not change adapter switch configuration.
