# v2.76.0 - Simulator Failure Scenarios

## Summary

Added selectable mock Prism negative-path scenarios so operators can test failed and timed-out environment evidence without a Nutanix lab.

## Added

- `GET /api/prism/failure-scenarios`.
- `POST /api/prism/failure-scenarios/:id/activate`.
- Failed and timeout simulator task states.
- Admin Providers failure scenario controls.

## Guardrails

- Failure scenarios only affect simulator evidence.
- No real infrastructure is created, changed, or deleted.
