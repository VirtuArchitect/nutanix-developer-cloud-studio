# v2.36.0-production-cab-decision-record

## Highlights

- Adds production CAB decision records linked to production CAB handoff packets.
- Adds Admin Operations visibility for CAB decision, decision authority, condition list, rollback approval, and decision minutes evidence.
- Adds API coverage for listing and recording production CAB decision records.
- Extends smoke coverage across CAB handoff into the CAB decision panel.

## Safety Boundary

Production CAB decision records are evidence-only. The prototype does not promote, enable, or execute real adapters.

## Verification

- Unit tests cover missing CAB handoff packets and incomplete decision evidence.
- API tests cover list/create routes, audit events, disabled kill switch state, and `provisioningEnabled=false`.
- Playwright smoke validates the Admin Operations production CAB decision panel.
