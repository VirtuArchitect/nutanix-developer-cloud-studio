# v3.2.0 - Credential Provider Contract

Added a credential provider contract for future Vault or secret-manager integration without storing secrets in NDC Studio.

## Added

- `GET /api/credentials/provider-contracts`.
- `POST /api/credentials/provider-contracts`.
- Disabled/mock resolver evidence for reference validation and redaction rules.

## Safety

- Credential values are never resolved or persisted.
- Contracts emit references and redaction evidence only.
