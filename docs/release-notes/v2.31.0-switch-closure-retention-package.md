# v2.31.0-switch-closure-retention-package

Release date: 2026-07-07

## Summary

This release adds switch closure retention packages. Packages link switch outcome records to retained evidence manifests, lessons learned, rollback timer closure, and final audit retention confirmation before any future adapter promotion dossier can be considered.

## Added

- Switch closure retention package domain records.
- `GET /api/real-adapter/switch-closure-packages`.
- `POST /api/real-adapter/switch-closure-packages`.
- Admin Operations switch closure retention panel.
- E2E smoke coverage for preparing a closure package.

## Guardrails

- Closure packages are evidence-only.
- `provisioningEnabled` remains `false`.
- The prototype closes retained records only and does not execute adapter switch changes.
