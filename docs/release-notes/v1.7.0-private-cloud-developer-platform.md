# v1.7.0-private-cloud-developer-platform

Release date: 2026-07-06

## Summary

This release moves the prototype closer to an operational private-cloud developer platform candidate by adding API-backed lifecycle operation records and audit export readiness records.

## Added

- Private-cloud lifecycle operation records for extend, suspend, destroy, and rebuild.
- Admin Operations tab for lifecycle and audit workflows.
- Audit export readiness metadata with retention, redaction, and storage boundaries.
- API/client/server tests and E2E smoke coverage for the new operations workflow.

## Guardrails

- Lifecycle operations are records for operator review only.
- Audit export is metadata only until external storage is configured.
- No Nutanix mutation APIs are called.
- `provisioningEnabled` remains `false`.
