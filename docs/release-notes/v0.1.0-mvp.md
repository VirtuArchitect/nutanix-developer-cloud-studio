# v0.1.0-mvp

First public MVP release of Nutanix Developer Cloud Studio.

## Highlights

- Clickable developer portal dashboard.
- Golden-path app catalog and template details.
- Create-environment workflow for VM, Kubernetes, database, storage, and AI endpoint targets.
- Simulated policy, cost, compliance, integration, and provisioning checks.
- Environment status timeline with mock provisioning states.
- AI endpoint approval pause behavior.
- Platform admin view with governance, integration health, and readiness notes.
- Mock Nutanix concepts for NCI, NKP, NDB, NUS, NCM, and NAI.
- Local persistence for requested environments and template governance edits.
- Unit tests for provisioning service behavior.
- GitHub issue and pull request templates.
- Configurable Obsidian documentation sync.

## Disclaimer

This release is an independent clickable MVP prototype. It does not currently provision real Nutanix infrastructure, call real Nutanix APIs, or include real credentials.

## Verification

- `npm.cmd run test`
- `npm.cmd run build`

## Notes

Use this release to validate the product thesis, gather feedback, and decide which real integration should be implemented first.
