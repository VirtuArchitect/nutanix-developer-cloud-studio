# v2.61.0 - Production Execution Archive Recovery Drill Record

Released: 2026-07-07

## Summary

This release adds production execution archive recovery drill records to the on-prem/control-plane readiness path. The record links to a production execution archive retrieval validation record and captures recovery drill evidence before any future archive recovery acceptance phase.

## Added

- Production execution archive recovery drill records linked to archive retrieval validation records.
- API endpoints for listing and recording archive recovery drill records.
- Admin Operations archive recovery drill panel.
- Drill owner, recovery scenario, elapsed recovery proof, restored artifact review, and drill sign-off checks.
- Unit, API, and smoke coverage for archive recovery drill readiness.

## Verification

- Missing archive retrieval validation records are rejected.
- Blocked or incomplete archive retrieval validation evidence keeps archive recovery drill records blocked.
- Drill evidence defaults produce a ready record only when the linked archive retrieval validation record is ready.
- Records keep `provisioningEnabled=false`.
- Records keep the provider kill switch disabled.

## Security Notes

- The prototype remains non-mutating.
- No credential, API token, or secret material is stored in the archive recovery drill record.
- The record stores recovery scenario, elapsed recovery proof, restored artifact review, drill sign-off, and ownership metadata only.
