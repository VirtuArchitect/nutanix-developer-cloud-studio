# v2.9.0-provider-release-gate-evidence

## Summary

This release adds provider release gate evidence for NCI, NKP, NDB, NUS, and NAI. The gate rolls up the evidence required before a provider adapter can be considered for controlled lab release while keeping all real adapter execution disabled.

## Added

- Provider release gate records for NCI, NKP, NDB, NUS, and NAI.
- API endpoints for listing and recording provider release gate reviews.
- Admin Control Plane provider release evidence envelope.
- Checks for lab scope, credential references, VM lifecycle proof, adapter enablement, provider contract review, provider preflight, audit export, release approver, and disabled real-adapter switch state.
- Unit, API, client, and smoke coverage for the provider release gate path.

## Verification

- `npm.cmd run test -- server/providerReleaseGate.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Phase gate must pass before promotion.

## Security Notes

- Provider release gates are evidence-only records.
- Real Nutanix adapter execution remains disabled.
- Missing authorization, evidence, approver data, or disabled-switch proof blocks release review.
