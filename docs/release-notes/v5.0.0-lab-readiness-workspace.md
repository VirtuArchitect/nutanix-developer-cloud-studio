# v5.0.0 - Lab Readiness Workspace

This release adds the first controlled mock-to-lab transition record.

## Added

- `GET /api/lab-transition/readiness-workspaces`
- `POST /api/lab-transition/readiness-workspaces`
- Admin Providers panel for lab readiness workspace evidence.
- Readiness rollup for mock Prism, disabled read-only adapter, runtime policy, pilot session, evidence review, rollback drill, and blockers.

## Safety

- Workspaces are evidence-only.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
