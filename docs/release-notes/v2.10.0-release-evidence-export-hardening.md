# v2.10.0-release-evidence-export-hardening

## Summary

This release adds release evidence export manifests linked to provider release gates. Exports are redacted metadata-only records for operations, security, and platform owner review.

## Added

- Release evidence export records linked to provider release gates.
- API endpoints for listing and preparing release evidence exports.
- Admin Operations panel for release evidence export history.
- Redacted JSON manifest metadata with provider, gate status, check counts, blocked operations, kill switch state, and evidence references.
- Unit, API, client, and smoke coverage for the export path.

## Verification

- `npm.cmd run test -- server/releaseEvidenceExport.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Phase gate must pass before promotion.

## Security Notes

- Release evidence exports contain references and metadata only.
- Inline auth material is redacted before manifest persistence.
- Real Nutanix adapter execution remains disabled.
