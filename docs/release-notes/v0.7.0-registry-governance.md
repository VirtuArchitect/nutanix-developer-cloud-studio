# v0.7.0-registry-governance

`v0.7.0-registry-governance` adds API-backed governance for template versions, policy bundles, and resource profile publication controls.

## Highlights

- Added template registry entries with draft, pending approval, published, and deprecated states.
- Added reusable policy bundles for sandbox, data protection, AI safety, and regulated audit controls.
- Added submit, approve, deprecate, and restore actions for template registry entries.
- Added matching governance actions for AHV, NKP, NDB, NUS, and NAI resource profiles.
- Added Admin Templates tab sections for image/profile catalog, template registry, policy bundles, and editable template governance.
- Added audit evidence for registry and resource profile lifecycle actions.
- Expanded unit, API, client, and Playwright smoke coverage.

## Important Notes

This remains a prototype release. Registry approval and published status are simulated governance records only. They do not provision real Nutanix infrastructure, mutate live systems, or enable real adapter actions.
