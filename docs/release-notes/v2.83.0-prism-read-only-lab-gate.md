# v2.83.0 - Prism Read-only Lab Gate

## Summary

Added a dedicated read-only Prism lab gate for future live-readiness review.

## Added

- `GET /api/prism/read-only-lab-gates`.
- `POST /api/prism/read-only-lab-gates`.
- Gate checks for active lab scope, reachable NCI config, credential reference, allowed list operations, excluded mutation operations, and disabled network execution.
- Admin Providers read-only gate panel.

## Guardrails

- Gates are evidence-only.
- Real Prism calls remain disabled.
