# v2.82.0 - Read-only Prism Adapter Scaffold

## Summary

Added a disabled-by-default read-only Prism adapter scaffold that constructs Prism list request plans without opening network connections.

## Added

- `GET /api/prism/read-only-adapter/diagnostics`.
- Fixture-only request plans for clusters, projects, images, subnets, categories, and VMs.
- Admin Providers diagnostics panel for read-only request plans.

## Guardrails

- `networkCallEnabled=false`.
- `provisioningEnabled=false`.
- Mutation operations remain explicitly blocked.
