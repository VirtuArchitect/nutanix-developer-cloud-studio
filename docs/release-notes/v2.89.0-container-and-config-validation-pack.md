# v2.89.0 - Container And Config Validation Pack

## Added

- Container/config validation manifest at `/api/deployment/config-validation`.
- `npm run validate:container-config` for Docker, Compose, env, runbook, rollback, and disabled adapter guardrail checks.
- Admin Operations panel for config validation evidence.

## Guardrails

- Validation is metadata-only.
- No Nutanix endpoint is contacted.
- Real Prism adapter and real provisioning remain disabled.
