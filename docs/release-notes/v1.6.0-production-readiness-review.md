# v1.6.0-production-readiness-review

`v1.6.0-production-readiness-review` adds release-gate review records for on-prem/private-cloud readiness.

## What Changed

- Added production readiness review records.
- Added API endpoints for creating and listing readiness reviews.
- Added Admin Overview UI for running readiness reviews.
- Added checks for OIDC boundary, durable state, audit retention, lab authorization, VM lifecycle proof, AHV preflight, platform-service preflight coverage, and provisioning guardrail.
- Added audit evidence and automated coverage for readiness reviews.

## Safety Boundary

Readiness reviews are evidence-only records. They do not enable live provisioning, do not call Nutanix APIs, and keep `provisioningEnabled=false`.
