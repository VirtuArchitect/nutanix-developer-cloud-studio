# v7.3.0-signed-audit-export-manifests

## Summary

Adds signed audit export manifest evidence.

## Highlights

- Adds `GET /api/audit/signed-export-manifest`.
- Emits manifest digest, signing key reference, signature metadata, and verification checks.
- Keeps credential, endpoint, token, and payload values out of the signed manifest.

## Safety

Production signing requires an external key reference; signing secrets are not stored in repo or API state.
