# v2.40.0-production-execution-authorization-record

Release date: 2026-07-07

## Summary

This release adds production execution authorization records to the on-prem/control-plane readiness path. The record links to a production execution readiness record and captures final authorization evidence before any future change-ticket lock phase.

## Added

- Production execution authorization record domain model.
- API endpoints to list and create production execution authorization records.
- Admin Operations panel for production execution authorization evidence.
- API client helpers for execution authorization records.
- Tests for server validation, API route coverage, API client calls, and E2E smoke coverage.

## Guardrails

- Records are evidence-only.
- `provisioningEnabled` remains `false`.
- The provider kill switch remains disabled.
- The prototype does not promote, enable, or execute real adapters.

## Next

- `v2.41.0-production-change-ticket-lock-record` will add evidence-only change ticket lock records linked to execution authorization records.
