# v2.13.0-controlled-lab-dry-run-window

Release date: 2026-07-07

This release adds evidence-only controlled lab dry-run windows. Windows link the provider release runbook, release evidence export, lab scope, rollback owner, audit export readiness, and emergency stop contacts before any future lab activity can be proposed.

## Added

- Controlled lab dry-run window records with scheduled start and end times.
- API endpoints for listing and preparing dry-run windows.
- Admin Operations panel for linked evidence, readiness checks, and emergency stop contacts.
- Tests proving missing runbook, lab scope, rollback owner, or audit evidence blocks window readiness.

## Security Notes

- Window scheduling is evidence-only.
- Real Nutanix adapter execution remains disabled.
- Records store references and operational contacts only, not credentials or live endpoint secrets.
