# v2.97.0 - Live Read-Only Inventory Adapter

Added controlled read-only inventory pilot evidence records.

## Added

- `GET /api/prism/live-read-only-inventory-pilots`.
- `POST /api/prism/live-read-only-inventory-pilots`.
- Admin Providers panel for inventory pilot status and inventory summary.
- Inventory evidence for clusters, projects, images, networks, categories, and VMs.

## Safety

- Pilot records require runtime mode, authorization gate, closed runbook workflow, and sanitized fixture evidence.
- No Prism Central network call is made in this prototype release.
