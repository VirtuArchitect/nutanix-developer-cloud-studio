# v7.2.0-jwt-oidc-verification-boundary

## Summary

Adds JWT/OIDC verification boundary diagnostics.

## Highlights

- Adds `GET /api/auth/jwt-boundary`.
- Reports issuer, audience, JWKS, and trusted-header fallback readiness.
- Preserves trusted-header mode for reverse-proxy deployments.

## Safety

This is a verification posture endpoint; it does not accept or persist tokens.
