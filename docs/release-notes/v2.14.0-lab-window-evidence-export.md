# v2.14.0-lab-window-evidence-export

Release date: 2026-07-07

This release adds metadata-only evidence exports for controlled lab dry-run windows. Exports give operations, security, and platform reviewers a compact manifest with window timing, linked evidence references, readiness checks, emergency contacts, checksum, and disabled execution state.

## Added

- Lab window evidence export records linked to controlled lab dry-run windows.
- Redacted JSON manifest metadata for runbook, release export, lab scope, rollback owner, emergency contacts, checks, and disabled execution state.
- API endpoints for listing and preparing lab window evidence exports.
- Admin Operations export history for lab window evidence.
- Tests proving exports contain references and metadata only.

## Security Notes

- Lab window exports contain references and metadata only.
- Real Nutanix adapter execution remains disabled.
- No provider data, credentials, tokens, or endpoint secrets are exported.
