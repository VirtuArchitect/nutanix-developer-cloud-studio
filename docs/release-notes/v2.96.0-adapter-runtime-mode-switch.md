# v2.96.0 - Adapter Runtime Mode Switch

Added API-backed read-only adapter runtime mode records.

## Added

- `GET /api/prism/read-only-runtime-modes`.
- `POST /api/prism/read-only-runtime-modes`.
- Admin Providers panel for runtime mode selection.
- Fail-closed checks for simulated, fixture replay, and authorized read-only lab modes.

## Safety

- Unsafe mode changes fall back to simulated mode.
- `networkCallEnabled=false`, `realPrismCallsEnabled=false`, and `provisioningEnabled=false` remain enforced.
