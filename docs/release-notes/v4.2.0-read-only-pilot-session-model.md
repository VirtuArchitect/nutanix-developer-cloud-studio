# v4.2.0 - Read-Only Pilot Session Model

This release adds controlled read-only pilot session records.

## Added

- `GET /api/prism/read-only-pilot-sessions`
- `POST /api/prism/read-only-pilot-sessions`
- Admin Providers panel for pilot sessions.
- Operator, start/end window, approved gate, runtime mode, policy, and evidence-link fields.

## Safety

- Sessions are evidence records only.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
