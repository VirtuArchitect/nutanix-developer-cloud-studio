# v3.1.0 - Adapter Configuration Boundary

Added formal configuration boundaries for future real Prism Central read-only adapter settings.

## Added

- `GET /api/prism/real-read-only/config-boundaries`.
- `POST /api/prism/real-read-only/config-boundaries`.
- Admin Providers panel for endpoint reference, credential provider reference, CA validation mode, timeout, retry, operation allowlist, kill switch, and checks.

## Safety

- Stores references and guardrails only.
- Keeps `provisioningEnabled=false`, `networkCallEnabled=false`, and `realPrismCallsEnabled=false`.
