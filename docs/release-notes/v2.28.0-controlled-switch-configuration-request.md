# v2.28.0-controlled-switch-configuration-request

Release date: 2026-07-07

## Summary

This release adds controlled switch configuration request records. Requests link switch-state audit packages to operator confirmation, second reviewer acceptance, final dry-run proof, rollback timer, and retention evidence before any future operator handoff package can be considered.

## Added

- Controlled switch configuration request domain records.
- `GET /api/real-adapter/controlled-switch-requests`.
- `POST /api/real-adapter/controlled-switch-requests`.
- Admin Operations controlled switch request panel.
- E2E smoke coverage for recording a controlled switch request.

## Guardrails

- Controlled switch requests are evidence-only.
- `provisioningEnabled` remains `false`.
- The prototype does not change adapter switch state.
