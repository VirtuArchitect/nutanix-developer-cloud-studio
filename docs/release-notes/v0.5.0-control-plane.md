# v0.5.0-control-plane

`v0.5.0-control-plane` adds the provisioning control-plane backbone behind Nutanix Developer Cloud Studio. The release introduces queue records, worker actions, state transitions, retries, failures, and audit evidence while keeping real infrastructure provisioning disabled.

## Highlights

- API-backed control-plane job queue.
- State machine for queued, validating, awaiting approval, provisioning, ready, failed, and expired jobs.
- Mock orchestrator worker actions for advance, retry, and fail.
- Approval release automatically returns paused jobs to the queue.
- Admin Provisioning Control Plane panel with worker controls.
- Dashboard Control Plane Queue panel and active job tile.
- Audit events for queueing, transitions, retries, failures, and approval release.
- Tests and smoke coverage for control-plane queue behavior.

## Notes

This is still a safe control-plane skeleton. It does not call real Nutanix APIs and does not enable provisioning.
