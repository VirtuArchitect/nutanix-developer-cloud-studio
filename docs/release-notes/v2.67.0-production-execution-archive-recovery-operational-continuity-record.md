# v2.67.0 - Production Execution Archive Recovery Operational Continuity Record

Release date: 2026-07-07

`v2.67.0-production-execution-archive-recovery-operational-continuity-record` adds evidence-only production execution archive recovery operational continuity records linked to evidence custody closure records.

## Added

- Production execution archive recovery operational continuity record domain model.
- API routes for listing and recording archive recovery operational continuity records.
- Admin Operations archive recovery operational continuity panel.
- Client API helpers for fetching and creating operational continuity records.
- Smoke coverage for the archive recovery operational continuity panel.

## Evidence Checks

Operational continuity records require:

- Ready production execution archive recovery evidence custody closure record.
- Continuity owner.
- Runbook update reference.
- KPI baseline reference.
- Support handoff reference.
- Continuity sign-off reference.
- Disabled kill switch and `provisioningEnabled=false`.

## Verification

- Unit tests cover missing evidence custody closure records and incomplete operational continuity evidence.
- Server API tests cover create/list behavior and audit-event emission.
- Client API tests cover the archive recovery operational continuity endpoints.
- E2E smoke coverage exercises the Admin Operations archive recovery operational continuity workflow.
