# v2.25.0-real-adapter-lab-scope-activation

Release date: 2026-07-07

## Summary

This release adds real-adapter lab scope activation evidence. Activations link dispatch approvals to authorized lab scope, pentest completion, rollback ownership, bounded provider targets, and manual operator controls before any manual real-adapter switch review can be considered.

## Added

- Real-adapter lab scope activation records linked to dispatch approvals.
- API endpoints for listing and recording lab scope activations.
- Admin Operations lab scope activation panel.
- Authorized scope, pentest completion, rollback owner, bounded provider target, manual operator control, and disabled switch checks.
- Tests proving missing dispatch approvals or incomplete activation evidence block readiness.

## Security Notes

- Activations prepare manual switch review evidence only.
- `provisioningEnabled` remains `false`.
- Provider kill-switch state remains disabled.
- Activation APIs require the `Platform Admin` role.
- No real Nutanix adapter execution is enabled.
