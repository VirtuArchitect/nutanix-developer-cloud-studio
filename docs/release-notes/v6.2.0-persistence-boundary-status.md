# v6.2.0-persistence-boundary-status

## Summary

Adds repository boundary diagnostics to prepare the API for durable on-prem state.

## Highlights

- Adds `GET /api/storage/persistence-boundary`.
- Reports active repository mode, durability, repository interface, retention boundary, and future migration targets.
- Keeps the current `ApiStore` boundary compatible with JSON-file and future Postgres repository work.

## Validation

- API tests verify memory-mode diagnostics and disabled provisioning flags.
