# v2.21.0-controlled-lab-execution-evidence-ledger

Release date: 2026-07-07

This release adds evidence-only controlled lab execution evidence ledgers. Ledgers freeze immutable evidence references after dry-run checklist readiness while real adapter execution remains disabled.

## Added

- Controlled lab execution evidence ledger records linked to dry-run checklists.
- API endpoints for listing and recording evidence ledgers.
- Admin Operations evidence ledger panel.
- Immutable operator, observer, rollback, log, audit, and stop authority evidence references.
- Tests proving missing dry-run checklists or incomplete evidence block ledger readiness.

## Verification

- `npm.cmd run test -- server/controlledLabExecutionEvidenceLedger.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Full phase gate: `.\scripts\run-phase-gate.ps1 -TargetPhase v2.21.0-controlled-lab-execution-evidence-ledger`

## Security Notes

- Evidence ledgers are evidence-only.
- `provisioningEnabled=false` remains enforced.
- Real Nutanix adapter execution remains disabled.
- Ledger records store evidence references only, not provider data, credentials, tokens, or secrets.
