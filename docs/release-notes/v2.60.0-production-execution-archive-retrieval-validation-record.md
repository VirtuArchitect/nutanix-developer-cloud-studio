# v2.60.0 - Production Execution Archive Retrieval Validation Record

Released: 2026-07-07

## Summary

This release adds production execution archive retrieval validation records to the on-prem/control-plane readiness path. The record links to a production execution readiness archive handoff record and captures retrievability evidence before any future archive recovery drill phase.

## Added

- Production execution archive retrieval validation records linked to readiness archive handoff records.
- API endpoints for listing and recording archive retrieval validation records.
- Admin Operations archive retrieval validation panel.
- Retrieval operator, sample retrieval proof, checksum verification, access audit, and recovery SLA witness checks.
- Unit, API, and smoke coverage for archive retrieval validation.

## Verification

- Missing readiness archive handoff records are rejected.
- Blocked or incomplete readiness archive handoff evidence keeps archive retrieval validation records blocked.
- Retrieval evidence defaults produce a ready record only when the linked readiness archive handoff record is ready.
- Records keep `provisioningEnabled=false`.
- Records keep the provider kill switch disabled.

## Security Notes

- The prototype remains non-mutating.
- No credential, API token, or secret material is stored in the archive retrieval validation record.
- The record stores retrieval proof references, checksum references, access audit references, recovery SLA witness references, and ownership metadata only.
