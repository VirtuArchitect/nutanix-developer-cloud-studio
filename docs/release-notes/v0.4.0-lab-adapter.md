# v0.4.0-lab-adapter

`v0.4.0-lab-adapter` introduces the first read-only lab adapter pilot shape for Nutanix Developer Cloud Studio. The phase is intentionally conservative: it models Prism Central/NCI discovery readiness while keeping all provisioning disabled.

## Highlights

- API-backed system status endpoint with `provisioningEnabled: false`.
- API-backed lab adapter snapshot list.
- Read-only discovery simulation for the NCI / Prism Central pilot path.
- Admin Lab Adapter Pilot panel with scope, guardrail, inventory count, and discovery action.
- Dashboard provisioning guardrail tile.
- JSON-file state backup/restore unit coverage.
- Hosted starter validation now checks lab adapter state and the provisioning-disabled guardrail.

## Notes

No real Nutanix APIs are called in this release. The lab adapter pilot is simulated and read-only. Real Prism Central integration should only be added after lab authorization, scope, endpoint, credential profile, and excluded mutation actions are documented.
