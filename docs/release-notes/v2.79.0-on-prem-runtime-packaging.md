# v2.79.0 - On-Prem Runtime Packaging

## Summary

Added runtime package validation for the on-prem starter bundle.

## Added

- `scripts/validate-runtime-package.ps1`.
- `npm run validate:runtime`.
- Checks for Dockerfile, Docker Compose, `.env.example`, healthcheck, static frontend serving, API startup, and disabled real adapter guardrails.

## Guardrails

- `NDC_PRISM_REAL_ADAPTER` must remain disabled in runtime packaging.
- The package validator does not require live Nutanix infrastructure.
