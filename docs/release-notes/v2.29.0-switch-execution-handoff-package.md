# v2.29.0-switch-execution-handoff-package

Release date: 2026-07-07

## Summary

This release adds switch execution handoff package records. Packages link controlled switch requests to operator run sheets, communications plans, observation windows, rollback-owner acceptance, and execution freeze proof before any future outcome record can be considered.

## Added

- Switch execution handoff package domain records.
- `GET /api/real-adapter/switch-handoff-packages`.
- `POST /api/real-adapter/switch-handoff-packages`.
- Admin Operations switch execution handoff panel.
- E2E smoke coverage for preparing a handoff package.

## Guardrails

- Handoff packages are evidence-only.
- `provisioningEnabled` remains `false`.
- The prototype does not execute adapter switch changes.
