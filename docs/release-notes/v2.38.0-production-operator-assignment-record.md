# v2.38.0-production-operator-assignment-record

Release date: 2026-07-07

## Summary

This release adds production operator assignment records to the on-prem/control-plane readiness path. The record links to a production implementation hold record and captures named execution ownership before any future execution readiness phase.

## Added

- Production operator assignment record domain model.
- API endpoints to list and create production operator assignment records.
- Admin Operations panel for production operator assignment evidence.
- API client helpers for operator assignment records.
- Tests for server validation, API route coverage, API client calls, and E2E smoke coverage.

## Guardrails

- Records are evidence-only.
- `provisioningEnabled` remains `false`.
- The provider kill switch remains disabled.
- The prototype does not promote, enable, or execute real adapters.

## Next

- `v2.39.0-production-execution-readiness-record` will add evidence-only execution readiness records linked to operator assignment records.
