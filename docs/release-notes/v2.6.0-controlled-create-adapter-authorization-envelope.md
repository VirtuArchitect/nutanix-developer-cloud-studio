# v2.6.0-controlled-create-adapter-authorization-envelope

`v2.6.0-controlled-create-adapter-authorization-envelope` defines the final authorization envelope for a future narrowly scoped AHV create adapter without enabling live mutation.

## Added

- Controlled-create authorization envelope records.
- `/api/vm-sandbox/controlled-create-authorization` endpoints for listing and recording envelope reviews.
- Checks for lab scope, rollback/destroy proof, controlled gate approval, lifecycle proof, adapter enablement, audit export, active pentest scope, and disabled real-mutation switches.
- Admin Control Plane panel showing blockers, allowed create fields, kill switch, and emergency stop procedure.
- Tests proving missing active pentest scope blocks live adapter authorization.

## Security Notes

This release is evidence-only. It does not call Prism Central or enable AHV create. The active pentest gate remains absent unless explicitly configured by a future authorized testing workflow.
