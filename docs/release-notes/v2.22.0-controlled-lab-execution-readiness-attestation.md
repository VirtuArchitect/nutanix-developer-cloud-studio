# v2.22.0-controlled-lab-execution-readiness-attestation

Release date: 2026-07-07

## Summary

This release adds evidence-only controlled lab execution readiness attestations. Attestations capture final platform, security, operations, rollback, and executive sponsor evidence after an evidence ledger is recorded while real adapter execution remains disabled.

## Added

- Controlled lab execution readiness attestation records linked to evidence ledgers.
- API endpoints for listing and recording readiness attestations.
- Admin Operations readiness attestation panel.
- Final attestation checks for platform owner, security reviewer, operations reviewer, rollback owner, and executive sponsor.
- Tests proving missing evidence ledgers or incomplete attestations block readiness.

## Security Notes

- Readiness attestations are evidence-only and do not enable real Nutanix adapter execution.
- `provisioningEnabled` remains `false`.
- Provider kill-switch state remains disabled.
- Attestation APIs require the `Platform Admin` role.
