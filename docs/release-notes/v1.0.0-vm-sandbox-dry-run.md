# v1.0.0-vm-sandbox-dry-run

`v1.0.0-vm-sandbox-dry-run` adds the first AHV VM sandbox planning path while keeping all infrastructure mutation disabled.

## Highlights

- Added VM sandbox dry-run plan model.
- Added API endpoints for creating and listing dry-run plans.
- Validates approved AHV image, published VM template, project, cluster, network, lifecycle category, quota, expiry, cost, and approval evidence.
- Adds rollback-plan evidence without performing any rollback or provisioning actions.
- Adds Admin Control Plane UI for generating and reviewing dry-run plans.
- Expands unit, API, client, and Playwright smoke coverage.

## Important Notes

This release is dry-run only. It does not call Prism Central to create, clone, power on, power off, resize, delete, or tag any VM. Real controlled provisioning remains gated to a future authorized phase.
