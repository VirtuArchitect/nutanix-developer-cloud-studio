# v2.16.0-lab-execution-proposal-envelope

Release date: 2026-07-07

This release adds evidence-only lab execution proposal envelopes. The envelope rolls up accepted lab evidence reviews with the underlying runbook, dry-run window, window export, lab scope, rollback owner, audit export readiness, emergency contacts, and disabled real-adapter state before any future controlled lab execution proposal can be considered.

## Added

- Lab execution proposal envelope records linked to lab evidence reviews.
- API endpoints for listing and recording lab execution proposal envelopes.
- Admin Operations proposal readiness panel.
- Proposal readiness checks for lab scope, runbook, dry-run window, evidence export, review acceptance, rollback owner, audit export, emergency contacts, and disabled execution.
- Tests proving missing or rejected review evidence blocks proposal readiness.

## Security Notes

- Proposal envelopes are evidence-only.
- `provisioningEnabled=false` remains enforced.
- Real Nutanix adapter execution remains disabled.
- Envelope evidence stores references and metadata only, not provider data, credentials, tokens, or secrets.
