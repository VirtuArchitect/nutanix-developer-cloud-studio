# v2.17.0-lab-execution-proposal-export

Release date: 2026-07-07

This release adds metadata-only lab execution proposal exports. Proposal exports produce redacted JSON manifest records linked to lab execution proposal envelopes for final operations and security review.

## Added

- Lab execution proposal export records linked to proposal envelopes.
- API endpoints for listing and preparing proposal exports.
- Admin Operations proposal export history panel.
- Redacted manifest metadata for proposal checks, evidence references, rollback owner, emergency contacts, kill switch state, and disabled execution state.
- Tests proving proposal exports contain references and metadata only.

## Verification

- `npm.cmd run test -- server/labExecutionProposalExport.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Full phase gate: `.\scripts\run-phase-gate.ps1 -TargetPhase v2.17.0-lab-execution-proposal-export`

## Security Notes

- Proposal exports are metadata-only.
- `provisioningEnabled=false` remains enforced.
- Real Nutanix adapter execution remains disabled.
- Proposal exports redact embedded URL credentials and query-string auth material before persistence.
