# v2.98.0 - Adapter Observability And Redaction

Added redacted read-only adapter observability records.

## Added

- `GET /api/prism/read-only-observability`.
- `POST /api/prism/read-only-observability`.
- Admin Operations panel for traces, latency summaries, blocked mutation counts, and audit evidence.
- Redaction summary for endpoint, credential profile, authorization header, and token fields.

## Safety

- Observability records are metadata-only.
- Real endpoints, credentials, authorization headers, tokens, and customer payloads are not stored.
