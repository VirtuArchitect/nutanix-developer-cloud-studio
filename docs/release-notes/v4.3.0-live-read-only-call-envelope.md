# v4.3.0 - Live Read-Only Call Envelope

This release adds non-executing call envelopes for future Prism read-only list operations.

## Added

- `GET /api/prism/live-read-only-call-envelopes`
- `POST /api/prism/live-read-only-call-envelopes`
- Admin Providers panel for call-envelope evidence.
- Method, path, timeout, retry, request ID, redaction fields, expected response shape, and `executionEnabled=false` per operation.

## Safety

- Envelopes model future calls but cannot execute them.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
