# v4.0.0 - Authorized Read-Only Lab Pilot Gate

Added the final non-executing operator gate before any future live read-only pilot can be considered.

## Added

- `GET /api/prism/authorized-read-only-pilot-gates`.
- `POST /api/prism/authorized-read-only-pilot-gates`.
- Required approvers, operator acknowledgements, stop conditions, and readiness checks.

## Safety

- The gate does not enable provisioning, inventory import, HTTP execution, or real Prism calls.
