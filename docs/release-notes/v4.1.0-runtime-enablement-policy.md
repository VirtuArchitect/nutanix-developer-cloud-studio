# v4.1.0 - Runtime Enablement Policy

This release adds formal runtime policy evidence for the future `NDC_PRISM_READONLY_HTTP_ENABLED` flag.

## Added

- `GET /api/prism/read-only-runtime-policies`
- `POST /api/prism/read-only-runtime-policies`
- Admin Providers panel for runtime policy records.
- Required approvals, allowed environments, expiry, simulated rollback mode, and emergency stop evidence.

## Safety

- The flag is documented only.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
