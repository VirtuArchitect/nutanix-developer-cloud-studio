# v5.3.0 - Lab Connection Dry-Run Console

This release adds an operator-facing dry-run console record for the future lab pilot.

## Added

- `GET /api/lab-transition/dry-run-consoles`
- `POST /api/lab-transition/dry-run-consoles`
- Admin Providers panel for lab dry-run console records.
- Endpoint reference, credential provider reference, allowed operations, expected requests, expected responses, blocked mutations, and rollback path.

## Safety

- Dry-run console records do not open sockets.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
