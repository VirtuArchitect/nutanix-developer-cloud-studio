# v3.5.0 - Authorized Lab Connection Dry Run

Added an operator-controlled dry run for future authorized Prism Central read-only lab connection review.

## Added

- `GET /api/prism/authorized-lab-dry-runs`.
- `POST /api/prism/authorized-lab-dry-runs`.
- Validation for configuration boundary, credential reference, TLS mode, endpoint allowlist, offline replay, production decision gate, redaction, and audit evidence.

## Safety

- Dry runs do not open sockets, resolve credentials, or call Prism Central.
- Real provisioning and real Prism calls remain disabled.
