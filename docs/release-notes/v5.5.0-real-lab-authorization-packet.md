# v5.5.0 - Real Lab Authorization Packet

This release adds the final request-ready packet for authorized Nutanix lab access.

## Added

- `GET /api/lab-transition/real-lab-authorization-packets`
- `POST /api/lab-transition/real-lab-authorization-packets`
- Admin Providers panel for real lab authorization packet records.
- Required lab details, Prism Central endpoint reference requirements, credential vault requirements, network boundary, approval owners, rollback owner, pentest scope evidence, and go/no-go checklist.

## Safety

- Authorization packets are request-ready evidence only.
- Live HTTP, provisioning, credential resolution, inventory import, and real Prism calls remain disabled.
