# v2.24.0-execution-broker-dispatch-approval

Release date: 2026-07-07

## Summary

This release adds execution broker dispatch approvals. Dispatch approvals are non-executing evidence records that bind broker queue records to rollback proof, pentest evidence, operator approver, and dispatch-window references while real adapter execution remains disabled.

## Added

- Execution broker dispatch approval records linked to broker queue records.
- API endpoints for listing and recording dispatch approvals.
- Admin Operations dispatch approval panel.
- Rollback proof, operator approver, pentest evidence, dispatch window, and disabled execution checks.
- Tests proving missing broker records or incomplete dispatch evidence block readiness.

## Security Notes

- Dispatch approvals are non-executing evidence records.
- `provisioningEnabled` remains `false`.
- Provider kill-switch state remains disabled.
- Dispatch approval APIs require the `Platform Admin` role.
- Real adapter execution remains disabled.
