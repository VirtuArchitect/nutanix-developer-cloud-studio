# v1.8.0-on-prem-packaging-hardening

Release date: 2026-07-06

## Summary

This release hardens the on-prem starter by adding operator validation and JSON state backup/restore tooling around the existing containerized API and static UI bundle.

## Added

- On-prem configuration validator.
- JSON state backup and restore scripts.
- Backup/restore smoke script.
- Phase gate coverage for on-prem validation and backup/restore.
- Compose environment guardrails for repository mode, audit retention, rate limiting, and disabled real Prism adapter state.
- Expanded deployment runbook with a deployment matrix and security checklist.

## Guardrails

- The starter still uses JSON state unless a future approved database phase is introduced.
- Backup tooling validates state shape but is not a production database backup system.
- Real Nutanix mutation adapters remain disabled.
- `provisioningEnabled` remains `false`.
