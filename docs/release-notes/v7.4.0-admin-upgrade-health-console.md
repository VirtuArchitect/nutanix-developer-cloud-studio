# v7.4.0-admin-upgrade-health-console

## Summary

Adds Admin upgrade health for on-prem operations.

## Highlights

- Adds `GET /api/admin/upgrade-health`.
- Surfaces repository mode, schema version, auth posture, audit signing status, config drift, and upgrade blockers.
- Adds an Admin Operations UI panel for the v7 foundation.

## Safety

The console is read-only and does not run migrations or change infrastructure.
