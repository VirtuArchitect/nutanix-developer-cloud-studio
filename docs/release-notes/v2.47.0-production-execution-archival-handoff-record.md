# v2.47.0-production-execution-archival-handoff-record

Release date: 2026-07-07

## Summary

This release adds production execution archival handoff records to the on-prem/control-plane readiness path. The record links to a production execution closure packet record and captures archival evidence before any future execution retention attestation phase.

## Added

- Production execution archival handoff record domain model.
- API endpoints to list and create production execution archival handoff records.
- Admin Operations panel for production execution archival handoff evidence.
- API client helpers for execution archival handoff records.
- Tests for server validation, API route coverage, API client calls, and E2E smoke coverage.

## Guardrails

- Records are evidence-only.
- `provisioningEnabled` remains `false`.
- The provider kill switch remains disabled.
- The prototype does not promote, enable, or execute real adapters.

## Next

- `v2.48.0-production-execution-retention-attestation-record` will add evidence-only execution retention attestation records linked to archival handoff records.
