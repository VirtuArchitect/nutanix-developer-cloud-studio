# v2.12.0-controlled-lab-release-runbook

Release date: 2026-07-07

This release adds evidence-only controlled lab release runbooks before any future adapter release proposal can be considered. Runbooks link to provider release readiness and record required human sign-offs, stop conditions, and escalation contacts.

## Added

- Controlled lab release runbook records linked to provider release readiness.
- API endpoints for listing and preparing runbooks.
- Admin Operations panel for sign-offs, stop conditions, escalation contacts, and disabled execution evidence.
- Tests proving missing sign-offs block runbook completion.

## Verification

- `npm.cmd run test -- server/controlledLabReleaseRunbook.test.ts server/apiServer.test.ts src/services/cloudStudioApi.test.ts`
- `npm.cmd run build:all`
- Full phase gate: `.\scripts\run-phase-gate.ps1 -TargetPhase v2.12.0-controlled-lab-release-runbook`

## Security Notes

- Runbook records are evidence-only.
- Real Nutanix adapter execution remains disabled.
- No credentials, tokens, endpoint secrets, or pentest report contents are stored.
