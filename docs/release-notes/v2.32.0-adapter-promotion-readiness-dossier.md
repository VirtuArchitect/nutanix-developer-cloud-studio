# v2.32.0-adapter-promotion-readiness-dossier

Release date: 2026-07-07

## Summary

This release adds adapter promotion readiness dossiers. Dossiers assemble retained switch closure evidence, monitoring plans, rollback drill confirmation, and security acceptance before any future production adapter authorization packet can be considered.

## Added

- Adapter promotion readiness dossier domain records.
- `GET /api/real-adapter/adapter-promotion-dossiers`.
- `POST /api/real-adapter/adapter-promotion-dossiers`.
- Admin Operations adapter promotion dossier panel.
- E2E smoke coverage for preparing a promotion dossier.

## Guardrails

- Promotion dossiers are evidence-only.
- `provisioningEnabled` remains `false`.
- The prototype does not promote or enable real adapters.
